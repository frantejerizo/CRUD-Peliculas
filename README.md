# 🎥 Catálogo de Películas - CRUD App

Este proyecto es una aplicación en **JavaScript** que permite gestionar un catálogo básico de películas mediante operaciones CRUD: **Crear, Leer, Actualizar y Eliminar**. Es una práctica para consolidar los conceptos aprendidos en clase sobre desarrollo web.

## 🚀 Características

- **Crear**: Permite agregar una nueva película al catálogo con las siguientes propiedades:
  - `id`: Generado automáticamente (único para cada película).
  - `titulo`: Nombre de la película.
  - `director`: Nombre del director.
  - `anio`: Año de estreno.
  - `descripcion`: Sinopsis de la película.
  - `imagen`: URL para la imagen de la película.
  - `genero`: Categoría (acción, comedia, drama, etc.).

- **Leer**: Redirige al usuario a una página de detalles para visualizar toda la información de la película.

- **Actualizar**: Permite modificar los datos de una película (excepto el `id`) a través de un modal.

- **Eliminar**: Elimina una película del catálogo utilizando su `id`.
